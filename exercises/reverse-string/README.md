# Reverse String

Reverse a string

For example:
input: "cool"
output: "looc"


To run the tests:

```sh
$ pub run test
```

For more detailed info about the Dart track see the [installation](http://exercism.io/languages/dart/installation) and [testing](http://exercism.io/languages/dart/tests) pages.
